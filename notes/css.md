# CSS 学习记录

## z-index:
设置视图层级，值越大离用户越近，会将值比较低的视图遮盖；类似于移动端后添加的视图
## position
这些修饰都是相对于父元素中而言
### releative
相对于原来默认位置， 原本分配的空间还是保留
### absolute
相对于浏览器视窗的位置
### fixed
固定位置

## display
### block
前后带有换行符，修饰元素会垂直依次显示
### inline
前后不带换行符，在水平方向显示，自身水平方向显示不下会换行;设置之后宽高自适应
### inline-block
类似 `inline` ，但是在所在行显示不下会自动换行

### flex

标注为 `flex` 之后，同时指定 `flex-flow` 或者 `flex-drection`  视图内部的子视图最好是确定
